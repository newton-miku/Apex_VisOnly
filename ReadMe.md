颜色解释

```c++
/*可见为绿色
不可见为红色
倒地敌人为蓝色*/
可见敌人为绿色
不可见敌人为护甲颜色（无甲/空甲也为白色）
倒地敌人为蓝色（闪烁）
```

使用方法

```
打开寄汤入口.bat查看提示即可
```
